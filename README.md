## 安装

**Step 1.** 确保你的机器全局安装 `Node.js`, `yarn`

**Step 2.** 克隆这个库来启动你的项目

```
yarn
yarn start
```

**Step 3.** 本地地址 `http://localhost:8888`

### 本地开发模块

